Clase 09: Creación de Gemas y Estrategias "Paso a Paso"

13 de Noviembre, 2025 | Semana 02

Objetivo de la Sesión

Esta sesión fue eminentemente práctica e interactiva. El objetivo principal fue asegurar que todos los participantes tuvieran acceso correcto a Gemini y comenzaran a crear sus propias "Gemas" (asistentes personalizados). Además, se analizó un caso de éxito real sobre migración de datos asistida por IA.

El Mundo de las "Gemas"

Una "Gema" es una versión personalizada de Gemini instruida para una tarea específica. Durante la clase, revisamos los componentes esenciales para crear una:

1. Descripción e Instrucciones

La descripción explica brevemente qué hace la gema. Las instrucciones son el "cerebro" donde defines cómo debe comportarse.

2. La "Varita Mágica"

Gemini ofrece una herramienta (varita mágica) que ayuda a redactar un prompt inicial basado en una idea simple.

3. Base de Conocimiento

Es fundamental para dar contexto. Puedes subir archivos (Excel, PDF, TXT) que la IA usa como referencia específica que no encontraría en internet.

Tip de Ahorro: Si usas la versión gratuita de Gemini, selecciona modelos "Flash" o "Mini". Consumen menos créditos y son ideales para tareas que no requieren razonamiento ultra-profundo.

Laboratorio: Ideas en Acción

Los participantes desarrollaron prototipos de Gemas para resolver problemas reales de su día a día:

  • Análisis de Jobs (JCL): Raúl Alarcón creó una gema para analizar y optimizar código JCL cargando archivos TXT.
  • Validador de JCL ("Chequea JCL"): Pedro Concha diseñó un asistente para revisar comandos no obvios en los JCL.
  • Conversor TXT a Excel: Crizla Urrutia trabajó en una gema para estructurar archivos de texto plano en columnas de Excel.
  • Validador de Procesos Masivos: Rafael Valdes propuso una gema para buscar strings repetidos y agrupar archivos por sistema para detectar inconsistencias en clasificaciones.

Estudio de Caso: El Poder del "Paso a Paso"

Se presentó un caso real de migración en BCI: transformar 150 procesos antiguos en 450 archivos nuevos de configuración.

El Problema

El intento inicial de pedirle a la IA que generara los 3 archivos de salida simultáneamente resultó en caos e inconsistencias.

La Solución Ganadora

La clave del éxito fue cambiar la estrategia de "dame todo ya" a un enfoque controlado:

  1. Plantillas Limpias: Usar esqueletos vacíos en la base de conocimiento para evitar datos "alucinados".
  2. Iteración Secuencial: Generar un archivo a la vez.
  3. Validación Humana: El experto revisa entre cada paso.

La Fórmula del Éxito

Resultado: Un proyecto estimado para 8 meses se aceleró drásticamente gracias a esta colaboración inteligente.

Herramientas Avanzadas y Futuro

  • Google AI Studio: Un entorno más avanzado para desarrolladores que permite integrar APIs de Gemini directamente en aplicaciones web o prototipos.
  • Portal Corporativo con IA: Se mostró un prototipo de dashboard desarrollado sin código (solo con IA generativa) que integra herramientas diarias y acceso a Gemas.
  • IA Creativa (Suno): Se demostró el uso de IA para generar música y letras completas a partir de un simple prompt, mostrando el potencial creativo de estas herramientas.
